# PDF Chapters Directory

Place your Ramayana chapter PDF files in this directory with the following naming convention:

- `chapter1.pdf` - Bala Kanda
- `chapter2.pdf` - Ayodhya Kanda
- `chapter3.pdf` - Aranya Kanda
- `chapter4.pdf` - Kishkindha Kanda
- `chapter5.pdf` - Sundara Kanda
- `chapter6.pdf` - Yuddha Kanda
- `chapter7.pdf` - Uttara Kanda

The PDFs will be automatically loaded when users access each chapter.
