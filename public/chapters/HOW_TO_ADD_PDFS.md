# How to Add Ramayana PDFs

## Free PDF Sources

### 1. Project Gutenberg
- **URL:** https://www.gutenberg.org/ebooks/search/?query=ramayana
- **Description:** Free public domain Ramayana texts
- **How to use:** Download PDFs and save them in this folder

### 2. Internet Archive
- **URL:** https://archive.org/search.php?query=ramayana
- **Description:** Free Ramayana books and PDFs
- **How to use:** Download PDFs and save them in this folder

### 3. Sacred Texts
- **URL:** https://www.sacred-texts.com/hin/rama/index.htm
- **Description:** Free Ramayana translations
- **How to use:** Download PDFs and save them in this folder

## File Naming

Save your PDFs with these exact names:
- `chapter1.pdf` - Bala Kanda
- `chapter2.pdf` - Ayodhya Kanda
- `chapter3.pdf` - Aranya Kanda
- `chapter4.pdf` - Kishkindha Kanda
- `chapter5.pdf` - Sundara Kanda
- `chapter6.pdf` - Yuddha Kanda
- `chapter7.pdf` - Uttara Kanda

## Steps

1. Visit one of the free sources above
2. Download Ramayana PDFs (make sure they're free/public domain)
3. Rename them to chapter1.pdf, chapter2.pdf, etc.
4. Place them in this `public/chapters/` folder
5. Refresh your website - the PDFs will appear automatically!

## Note

Only use PDFs that are:
- Free to use
- Public domain
- Not copyrighted
- Legally available for download
